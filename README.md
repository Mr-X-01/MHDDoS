### Getting Started

**Requirements**

* [Python3][python3]
* Python-pip3
* requests
* PySocks
* cfscrape
* icmplib
* scapy
---


**Clone and Install Script**

```console
> git clone https://github.com/MHProDev/MHDDoS.git
> cd MHDDoS
> pip3 install -r requirements.txt
```

---

**Launch Script**

```console
> python3 start.py
```
